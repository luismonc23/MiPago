# MiPago

Plataforma de MiPago

## Uso

### Instal dependencies

Go to the downloaded repository folder and run:
```
npm install
```

### Run the app

```
npm run dev
```

App will be opened in browser at `http://localhost:8080/`

### Build app for production

```
npm run build
```

The output will be at `www/` folder
